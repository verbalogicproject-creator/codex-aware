# Codex Aware contributor guidance

- Product copy must remain competition-unaware.
- Context never implies authority.
- Do not report a directive as successful before an observed-effect ACK.
- Imported commands default to unclassified.
- Adapters and database rows cannot supply executable code.
- Never scan or persist `.env*`, credentials, raw databases, absolute paths, or unrestricted source bodies.
- Run `pytest`, `npm run typecheck`, `npm run build`, and plugin validation before release.
